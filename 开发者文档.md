# AimiliVPN 开发者文档

## 项目定位

AimiliVPN 是一个 Docker 化的 VPN/Xray 网关面板。正式运行路径以 Docker Compose 为准，宿主机只保留 Docker、持久化数据目录和 `ml` 管理入口；OpenVPN、Xray、代理网关、Web 面板、节点检测和路由同步都在容器内完成。

## 当前目录结构

```text
.
├── backend/              后端服务入口、API、状态、配置、Xray/OpenVPN 核心逻辑
├── cli/                  宿主机 ml 管理菜单
├── docker/               full 与 vpngate 两种镜像定义和容器 entrypoint
├── proxy/                本地 HTTP/SOCKS 代理网关
├── tests/                单元测试与部署契约测试
├── utils/                VPN、代理、网络探测通用工具
├── web/                  静态管理面板页面、样式和前端脚本
├── docker-compose.yml    Docker 正式部署编排
├── install-docker.sh     Docker 一键安装入口
└── install.sh            兼容旧命令，转交给 Docker 安装器
```

## 启动入口

容器内服务入口：

```bash
python -m backend.app.main
```

宿主机管理入口：

```bash
ml
ml status
ml logs
ml update
ml uninstall
```

默认 Web 管理端口为 `8787`，本地代理端口为 `7928`。配置和运行数据默认持久化在 `vpngate_data/`，容器内对应 `/data`。

## 已实现功能

- Docker 完整部署：一键安装、更新、卸载、重建镜像、保留持久化数据。
- Docker 网关边界：宿主机不再创建正式 OpenVPN/Xray 系统服务，网关能力运行在容器内。
- Web 管理面板：登录、隐藏路径、主题切换、服务状态、系统资源、日志和诊断入口。
- Xray Core 管理：安装检测、启动、停止、重启、配置保存、配置测试、运行日志尾部展示。
- 入站管理：创建订阅链接、创建独立入站节点、启停、删除、二维码、分享链接和订阅内容生成。
- 出站管理：VPNGate OpenVPN、Cloudflare WARP、自定义节点、订阅导入节点、JSON 配置节点。
- VPNGate 节点池：拉取节点、并发检测、延迟排序、国家筛选、IP 类型筛选、节点连接与断开。
- WARP 支持：注册 WARP 账号、生成 WARP 出站、测试 WARP 出口、删除 WARP 节点。
- 自定义出站测试：临时 Xray 进程测试指定出站节点出口 IP 和延迟。
- 策略路由：按入站、域名、IP、协议等条件绑定指定出站，并同步到 Xray 配置。
- 分层健康检测：API 连通性、节点池、OpenVPN 网卡、策略路由、本地代理出口检测。
- 流量统计：会话流量、本月累计、历史累计、24 小时趋势和 Xray 客户端流量。
- 功能开关：VPNGate、WARP、自定义节点按功能位展示；自定义节点能力保持开启。
- 安全清理：OpenVPN 账号密码不再落盘到旧认证文件，启动时自动清理旧凭据文件。

## 关键数据文件

运行时数据位于 `vpngate_data/` 或容器 `/data`：

```text
ui_auth.json              Web 面板账号、密码、隐藏路径、端口、域名证书配置
state.json                当前运行状态
nodes.json                VPNGate 节点池
xray_cfg.json             面板级 Xray 配置
xray_config.json          实际写给 Xray Core 的运行配置
subscription_links.json   入站订阅链接
subscription_nodes.json   入站节点
outbound_nodes.json       出站节点
routing_rules.json        路由规则
feature_flags.json        功能开关
traffic.json              流量累计
traffic_history.json      24 小时趋势数据
client_traffic.json       Xray 客户端流量
```

## 测试

当前测试使用 Python 标准库 `unittest`：

```bash
python3 -m unittest discover -s tests -v
```

测试覆盖 Docker 部署契约、前端内联动作、隐藏路径路由、功能开关、订阅节点、路由与出站、VPNGate/OpenVPN 诊断和代理解析工具。
